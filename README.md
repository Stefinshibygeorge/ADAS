# 🚘 Intelligent Vehicle Assistance and Autonomous Perception Suite

This repository contains a collection of projects focused on **Advanced Driver Assistance Systems (ADAS), Computer Vision, Embedded Systems, and Deep Learning for Intelligent Transportation**.  

The projects demonstrate practical implementations of autonomous driving concepts including:

- Adaptive Cruise Control
- Automatic Emergency Braking
- Lane Detection and Steering Estimation
- Regenerative Braking
- Pothole Detection using Deep Learning
- Sensor Fusion and Vehicle Perception

Each project folder contains its own detailed README with implementation-specific explanations, code structure, algorithms, and setup instructions.

# 🚗 Project Overview

## 1️⃣ Adaptive Cruise Control and ADAS Integration

A miniature **ADAS prototype** built using two Arduino-based modules communicating through the **I²C protocol**.

### Key Features
- Obstacle Detection
- Automatic Emergency Braking
- Speed/RPM Measurement
- Cruise Control
- Regenerative Braking
- Seatbelt Detection
- Adaptive Throttle Control

### Technologies
- Arduino
- Embedded C++
- Ultrasonic Sensors
- MCP4725 DAC
- I²C Communication

📁 Detailed implementation available inside:

```bash
/ADAS_Adaptive_Cruise_Control/README.md
```

## 🚘 Lane Detection and Steering Simulation

- Sliding Window Tracking
